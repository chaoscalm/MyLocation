<big><b>Λειτουργίες</b></big>

<b>My Location</b> βρίσκει την τοποθεσία της συσκευής σας με τους ακόλουθους τρόπους:

<b>-</b> <b>GPS</b> είναι συνήθως η πιο ακριβής μέθοδος. Αλλά η επιδιόρθωση θέσης, μπορεί να πάρει κάποιο χρόνο ή και να μην λειτουργήσει καθόλου λόγω απώλειας σήματος. <b>Κλείδωμα GPS</b> τρέχει μια μόνιμη υπηρεσία για να παραμείνει συνδεδεμένο με τους δορυφόρους.

Μπορείτε επίσης να δείτε τη λίστα των ορατών δορυφόρων με τα PRNs τους (μοναδικά αναγνωριστικά) και τα SNR (ποιότητα σήματος).

<b>-</b> <b>Πάροχος Τοποθεσίας Δικτύου</b> χρησιμοποιεί Wi-Fi ή κινητά για την εκτίμηση της τοποθεσίας. Στις συσκευές με εγκατεστημένες τις υπηρεσίες Google Play, το NLP συνήθως χρησιμοποιεί την υπηρεσία τοποθεσίας Google στο σύστημα υποστήριξης.

<b>-</b> <b>UnifiedNLP</b> είναι ένα API ανοικτού κώδικα που έχει χρησιμοποιηθεί για την ανάπτυξη πολλαπλών συστημάτων υποστήριξης NLP (https://github.com/microg/UnifiedNlp/wiki/Backends).

<b>Επιπλέον:</b>
<b>-</b> Οι συντεταγμένες τοποθεσίας, μπορούν να αντιγραφούν στο πρόχειρο ή να ανοιχτούν σε μια εφαρμογή χάρτη, εάν έχετε εγκατεστημένη.
<b>-</b> Υποστηρίζεται επίσης η εκκαθάριση δεδομένων A-GPS.
